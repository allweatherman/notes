    3) 자바스크립트
        - 사용자 동작에 반응하는 부분

3. 웹 개발 환경
    1) 웹 브라우저
        - 크롬, 엣지, 파이어폭스
    2) 웹 편집기
        - VS Code 
    3) 마크업하기 (Markup)
        - 웹 문서의 각 부분이 어떻게 표시되어야 하는지 내용마다 코드를 붙이는 것
        - 내용마다 붙이는 코드를 태그(tag)라고 함

4. HTML 기본 문서 만들기
    1) HTML 
        - HyperText Markup Language
        - 웹 문서를 만드는 언어
        - HyperText : 문서를 서로 연결해주는 링크
    2) HTML 문서의 기본 구조
        -<!DOCTYPE html>

        - <html> ~ </html>
        - <head> ~</head>
            - 웹 브라우저가 웹 문서를 해석하는데 필요한 정보를
        - <body> ~ </body>
            - 실제로 웹 브라우저 화면에 나타나는 내용
    
    3) 태그, 기억해두면 좋은 4가지
        - 태그는 소문자로 씀
        - 여는 태그와 닫는 태그를 정확히 입력함
        - 적당하게 들여쓰기를 함
        - 일부 태그는 속성과 함께 사용함
    
5. 시맨틱(semantic) 태그
    1) 문서 내용에는 영향을 주지 않으면서 웹 브라우저가 문서 구조를 파악하는 데 중요한 역할을 하는 태그임
    2) 주요 시멘틱 태그
        - <header>
            - 헤더 영역을 나타내는 태그
        - <nav>
            - 내비게이션 영역을 나타내는 태그
        - <main>
            - 핵심 콘텐츠를 담는 태그
            - 웹 문서에서 한번만 사용할 수 있음
            - 웹 문서마다 다르게 보여주는 내용으로 구성함
        - <article>
            - 독립적인 콘텐츠를 담는 태그
            - 여러 개 사용할 수 있고, 이 안에 <section>태그를 넣을 수 있음
        - <section>
            - 웹 문서에서 콘텐츠 영역을 나타내는 태그
            - 몇 개의 콘텐츠를 묶을 때 사용함
        - <asdie>
            - 사이드 바 영역을 나타내는 태그
        - <footer>
            - 푸터 영역을 나타내는 태그
            - 사이트 제작정보, 저작권 정보, 연락처 등을 넣음

6. HTML 주석 달기
    1) comment, 설명글
    2) <!-- 주석 내용 -->

7. 정리
    1) Element(요소) == Tag(태그)
        - 기본 형태
            - <태그이름>내용</태그이름>
            - <태그이름>
                내용
              </태그이름>
    
    2) 속성(Attribut)과 값(Value)
        - <태그이름 속성="속성값">내용</태그이름>
        - 태그(요소)의 기능을 확장하기 위해 '속성'을 사용함
    
    3) 부모 요소와 자식 요소
        - <부모요소>
            <자식요소></자식요소>
          </부모요소>
        - 요소의 내용으로 새로운 태그를 넣었을 때
            '부모와 자식'의 관계가 생김
    
    4) 빈 태그 (Empty tag)
        - 종료태그가 없는 태그를 지칭
        - <태그이름>
        - <태그이름/>
        - <태그이름 속성="속성값"/>
        - '빈 태그'의 경우 종료 태그가 없어 내용을 넣을 수 없으므로, 대부분의 빈태그는 속성과 값이 필요하

    5) 웹 브라우저가 인식할 문서의 범위를 설정
        - <html></html>
            - 문서의 전체 범위
        - <head></head>
            - 문서의 정보를 나타내는 범위
        - <body></body>
            - 문서의 구조를 나타내는 범위위
