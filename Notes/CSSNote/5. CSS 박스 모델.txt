1. 블록 레벨 요소와 인라인 레벨 요소
    1) block level
        - 태그를 사용해 요소를 삽입했을 때 혼자 한 줄을 차지하는 것을
          가리킴
        - 해당 요소의 너비가 100%라는 뜻임
            - 블록 레벨 요소의 왼쪽이나 오른쪽에 다른 요소가 올 수 없음
            - <h1>, <div>, <p>
    2) inline level
        - 한 줄을 차지하지 않음
        - 영역을 콘텐츠만큼만 차지함
            - 나머지 공간에는 다른 요소가 올 수 있음
            -<span>

2. 박스 모델 기본 구성
    1) 웹 문서의 블록 레벨 요소와 인라인 레벨 요소는 모두 박스 형태임
    2) 스타일 시트는 이렇게 박스 형태인 요소를 박스 모델box model 요소라고 함
    3) 박스 모델은 콘텐츠 영역, 
       박스와 콘첸츠 영역 사이의 여백인 패딩(padding), 박스의 테두리(border),
       그리고 여러 박스 모델 사이의 여백인 마진(margin) 등 요소로 구성됨

3. 콘텐츠 영역의 크기 지정하는 속성
    1) width, height
        - 속성값
            - 너비나 높이 값을 px이나 em 단위로 지정함
            - 백분율
                - 박스 모델을 포함하는 부모 요소를 기준으로 너비값, 높잇값을
                  백분율(%)로 지정함
            - auto
                - 박스 모델의 너비값, 높잇값이 콘텐츠 양에 따라 자동으로 
                  결정됨
    2) 박스 모델의 크기를 계산하는 속성
        - box-sizing
            - border-box : 패딩과 테두리를 포함해서 너비값을 지정함
            - content-box:
        - width, height 속성은 박스 모델에서 콘텐츠 영역의 크기를 가리킴
        - 웹 문서에 여러 가지 요소를 배치할 때 실제 박스 모델이 차지하는
          크기는 콘텐츠 영역 외에도 콘텐츠와 테두리 사이의 여백(패딩), 
          테두리 두께까지 계산해야 함

4. 박스 모델에 그림자 효과 주는 속성
    1) box-shadow
    2) 사진 주변에 그림자 효과 낼 수 있음

5. 박스 모델의 방향
    1) 상하좌우 4개 방향이 있어서 테두리, 마진, 패딩 등을 지정할 때 한꺼번에
       똑같이 또는 모두 다르게 지정할 수 있음
    2) 속성값이 top -> right -> bottom -> left 처럼 시계방향 순서로
       적용됨

6. 테두리 스타일 지정하는 속성
    1) border-style

7. 테두리 두께 지정하는 속성
    1) border-width
