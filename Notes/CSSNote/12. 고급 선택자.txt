1. 연결 선택자
    0) 하위 요소
        - 특정 요소를 기준으로 그 안에 포함된 요소 
    0) 자식 요소
        - 현재 요소를 기준으로 바로 한 단계 아래 요소
       손자 요소
        - 그 자식 요소의 한 단계 아래 요소
    0) 형제 요소
        - 부모 요소가 같은 경우 
        - 먼저 나오는 요소를 형 요소, 나중에 요소를 동생 요소


    1) 하위 선택자 (descendant selector)
        - 부모 요소에 포함된 하위 요소를 모두 선택
        - 자식 요소뿐만 아니라 손자 요소, 손자의 손자 요소 등
          모든 하위 요소까지 적용됨
        - 상위 요소와 하위 요소를 공백으로 구분함
        - 기본형
            - 상위 요소 하위 요소
            - 예) section p {...}
    
    2) 자식 선택자(child selector)
        - 하위 선택자와 다르게 자식 요소에만 스타일을 적용함
        - 기본형 
            - 부모요소 > 자식요소 
    3) 인접 형제 선택자 (adjacent selector)
        - 형제 요소 중에서 첫 번째 동생 요소만 선택하는 것
        - 기본형
            - 요소1 + 요소2
            - 예) h1 + p { color : blue; }
                 ---------
                 h1의 형제인 p 요소 중 첫번째 p

    4) 형제 선택자 (sibling selector)
        - 모든 형제 요소에 적용됨
        - 기본형
            - 요소1  ~  요소2
            - 예) h1 ~ p { color : blue; }
                 ---------
                  h1의 형제인 모든 p 요소 

2. 속성 선택자 
    1) 특정 속성이 있는 요소를 선택하는 [속성] 선택자
    - 대괄호( [] ) 사이에 원하는 속성을 입력하면 됨
    - 예 
        - a[href=]
    2) 특정 속성값이 있는 요소를 선택하는 [속성=속성값] 선택자
        - 예)
            - a[target = _blank]
            - a 요소 중에서 target 속성값이 _blank인 것만 선택함
    3) 여러 값 중에서 특정 속성값이 포함된 속성 요소를 선택하는 
       [속성~=값] 선택자
    4) 특정 속성값이 포함된 요소를 선택하는 [속성|=값] 선택자
        - 속성값은 한 단어로 일치해야 함
